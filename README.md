# Data Analysis Projects

This repository collects small-to-medium data analysis projects focused on exploratory analysis and interactive reporting using Power BI.

Included projects

- `data_jobs_dashboard_power_bi` — Data Jobs Dashboard (v1). A multi-page Power BI dashboard built from 2024 data science job postings. See the project's README for screenshots, the `.pbix` file, and a link to the Power BI service.

- `data_jobs_dashboard_2.0_power_bi` — Data Jobs Dashboard (v2). A redesigned, single-page Power BI dashboard with updated KPIs and streamlined navigation. See the project's README for details, visuals, and the `.pbix` file.

Resources

- `Resources/images/` — Animated GIFs and screenshots used by the project READMEs.

Quick start

1. Open either project folder in Power BI Desktop and open the corresponding `.pbix` file (`Data_Jobs_Dashboard.pbix` or `Data_Jobs_Dashboard_2.0.pbix`).
2. Explore the report pages, slicers, and visual interactions.
3. You can see the links to the published reports on Power BI service.
